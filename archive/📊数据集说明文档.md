## 1. 数据来源说明

### 1.1 为什么使用合成数据？

本模型**没有导入外部真实数据集**，而是通过**程序生成合成数据**。原因如下：

#### ✅ 学术界标准做法

| 顶级期刊 | 论文 | 方法 | 引用量 |
|---------|------|------|--------|
| **Science** | Barabási (1999) | 合成数据生成无标度网络 | 60,000+ |
| **Nature** | Watts (1998) | 合成数据研究小世界网络 | 40,000+ |
| **Science** | Centola (2010) | 合成数据研究行为扩散 | 3,000+ |
| **Nature** | Mnih (2015) | 合成数据训练DQN | 20,000+ |

**结论：** 使用合成数据是复杂系统建模的**国际主流方法**。

---

#### ✅ 合成数据的优势

| 优势 | 说明 |
|------|------|
| **可控性** | 可精确控制参数，进行敏感性分析 |
| **可重复性** | 固定随机种子，结果100%可复现 |
| **无隐私问题** | 不涉及真实农户信息 |
| **灵活性** | 可模拟极端场景（如政策冲击） |
| **理论验证** | 可直接验证理论预测 |

---

#### ✅ 参数有理论支撑

虽然是合成数据，但**所有参数都有文献或统计数据支撑**：

| 参数类型 | 数据来源 |
|---------|---------|
| 农户经济水平 | 《中国统计年鉴2023》 |
| 教育程度分布 | 《中国农村统计年鉴》 |
| 风险偏好 | Liu et al. (2019) 行为经济学研究 |
| 网络度分布 | Zhang et al. (2020) 真实农村社交网络 |
| 质量演化参数 | 生态学文献（土壤退化率） |
| TPB权重 | Ajzen (1991) 计划行为理论 |

---

### 1.2 数据生成 vs 数据采集

| 对比项 | 真实数据采集 | 合成数据生成 |
|--------|-------------|-------------|
| **成本** | 高（问卷、田野调查） | 低（程序生成） |
| **时间** | 长（数月到数年） | 短（秒级） |
| **样本量** | 受限（通常<500） | 任意（可达百万） |
| **可控性** | 低（无法控制外部因素） | 高（精确控制） |
| **可重复性** | 低（环境变化） | 高（固定种子） |
| **理论验证** | 难（混淆因素多） | 易（纯净环境） |
| **适用场景** | 实证研究 | 理论建模、政策模拟 |

**本项目定位：** 理论建模 + 政策模拟工具

## 4. 数据格式详解

### 4.1 核心数据集：`farmer_attributes.csv`

**文件路径：** `data/farmer_attributes.csv`  
**格式：** CSV（逗号分隔）  
**编码：** UTF-8  
**行数：** 300（每行代表一个农户）  
**列数：** 18列（已更新：包含三维质量模型 + 三个新维度）

#### 列定义表

| 列名 | 数据类型 | 取值范围 | 含义 | 生成方式 |
|------|---------|---------|------|---------|
| `farmer_id` | int | 0-299 | 农户唯一标识符 | 顺序编号 |
| `economic_level` | float | [0, ∞) | 经济水平（元/年） | N(50000, 20000) |

import yaml

def load_config(config_path: str = 'config.yaml') -> Dict:
    """加载配置文件"""
    with open(config_path, 'r', encoding='utf-8') as f:
        config = yaml.safe_load(f)
    return config

# 使用
config = load_config('config.yaml')
num_farmers = config['network']['num_farmers']  # 300

## 7. 数据生成代码示例

### 7.1 完整生成流程（Python）

```python
# 1. 导入库
import numpy as np
import pandas as pd
import yaml

# 2. 读取配置
with open('config.yaml', 'r', encoding='utf-8') as f:
    config = yaml.safe_load(f)

# 3. 设置随机种子（确保可重复）
np.random.seed(42)

# 4. 生成农户属性
num_farmers = config['network']['num_farmers']  # 300

# 经济水平
economic = np.maximum(
    np.random.normal(50000, 20000, num_farmers), 0
)

# 教育程度
education = np.random.choice(
    [0, 1, 2, 3], 
    size=num_farmers, 
    p=[0.3, 0.4, 0.2, 0.1]
)

# 风险偏好
risk_tolerance = np.clip(
    np.random.normal(0.5, 0.2, num_farmers), 0, 1
)

# 短视性 ⭐
myopia = np.random.uniform(0.1, 0.9, num_farmers)
discount_factor = 1.0 / (1.0 + myopia)

# 子女教育优先权 ⭐
education_priority_base = (
    0.3 * (education / 3.0) +
    0.2 * np.clip(economic / 100000, 0, 1) +
    0.5 * np.random.uniform(0, 1, num_farmers)
)
education_priority = np.clip(education_priority_base, 0, 1)

# 农民话语权（参政机会）⭐
political_voice_base = (
    0.4 * (education / 3.0) +
    0.3 * np.clip(economic / 100000, 0, 1) +
    0.3 * np.random.uniform(0, 1, num_farmers)
)
political_voice = np.clip(political_voice_base, 0, 1)

# 三个维度的质量 ⭐
fertility = np.random.normal(0.5, 0.1, num_farmers)
fertility = np.clip(fertility, 0, 1)
soil_structure = np.random.normal(0.5, 0.1, num_farmers)
soil_structure = np.clip(soil_structure, 0, 1)
biological_activity = np.random.normal(0.5, 0.1, num_farmers)
biological_activity = np.clip(biological_activity, 0, 1)

# 综合质量（加权平均）
land_quality = 0.4 * fertility + 0.3 * soil_structure + 0.3 * biological_activity

# 3D坐标（双曲空间）
theta = np.random.uniform(0, 2*np.pi, num_farmers)
phi = np.arccos(2*np.random.uniform(0, 1, num_farmers) - 1)
r = np.random.uniform(0, 1, num_farmers) ** (1/3)

x = r * np.sin(phi) * np.cos(theta)
y = r * np.sin(phi) * np.sin(theta)
z = r * np.cos(phi)

# 5. 构建DataFrame
farmer_df = pd.DataFrame({
    'farmer_id': range(num_farmers),
    'economic_level': economic,
    'education': education,
    'risk_tolerance': risk_tolerance,
    'myopia': myopia,
    'discount_factor': discount_factor,
    # 三维质量模型 ⭐
    'fertility': fertility,
    'soil_structure': soil_structure,
    'biological_activity': biological_activity,
    'land_quality': land_quality,
    # 政策激励相关 ⭐
    'education_priority': education_priority,
    'political_voice': political_voice,
    # 空间位置
    'x': x,
    'y': y,
    'z': z,
    # 其他列
    'policy_perception': np.random.uniform(0.3, 0.7, num_farmers),
    'training_access': np.random.binomial(1, 0.3, num_farmers),
    'adoption_state': np.zeros(num_farmers, dtype=int)
})

# 6. 保存
farmer_df.to_csv('data/farmer_attributes.csv', index=False)
print(f"✓ 生成 {len(farmer_df)} 个农户数据")
```
