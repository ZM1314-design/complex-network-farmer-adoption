## 1. network_final_state.png - 网络结构图

### 📐 图表内容

**3D可视化双曲几何网络结构**

- **X轴、Y轴、Z轴：** 农户在双曲空间中的三维坐标
- **节点（圆点）：**
  - 🟢 **绿色节点**：采纳绿色施肥的农户
  - ⚪ **灰色节点**：使用传统施肥的农户
- **节点大小：** 表示农户的**度数**（连接数），度数越大节点越大
- **连线（蓝色细线）：** 表示农户之间的社交关系

### 🎯 科学意义

1. **网络拓扑结构：**
   - 展示双曲几何网络的**无标度特性**
   - 可以看到少数**Hub节点**（大节点，连接很多农户）
   - 大多数节点度数较小

2. **采纳状态分布：**
   - 绿色节点 = 采纳绿色施肥
   - 灰色节点 = 传统施肥
   - 可以看到采纳行为在网络中的**空间分布**

3. **网络效应：**
   - 观察采纳者是否**聚集**在一起（网络传播效应）
   - 观察Hub节点是否更容易采纳（影响力效应）

### 📊 关键信息

- **网络规模：** 300个节点（农户）
- **连接数：** 约1500-2400条边
- **平均度数：** 约8-10
- **采纳率：** 最终状态的采纳率

## 2. degree_distribution.png - 度分布图

### 📐 图表内容

**2个子图：**

**左图：度分布直方图**
- **X轴：** 度数 k（农户的连接数）
- **Y轴：** 频率（有多少农户具有该度数）
- **柱状图：** 显示不同度数的农户数量

**右图：对数-对数图（验证幂律）**
- **X轴：** 度数 k（对数刻度）
- **Y轴：** 概率 P(k)（对数刻度）
- **散点：** 实际数据点
- **拟合线：** 理论幂律分布 P(k) ~ k^(-2.5)

### 🎯 科学意义

1. **验证无标度网络：**
   - 真实社交网络符合**幂律分布**
   - 少数农户连接很多（Hub节点）
   - 大多数农户连接较少

2. **理论验证：**
   - 理论预测：P(k) ~ k^(-2.5)
   - 实际测量：验证模型是否正确生成无标度网络
   - **误差 < 5%** 表示模型准确

3. **网络特性：**
   - **幂律指数 γ：** 约2.5（与理论值一致）
   - **Hub节点：** 度数最大的10%农户

### 📊 关键信息

- **幂律指数：** γ ≈ 2.5
- **最大度数：** k_max ≈ 20-30
- **平均度数：** <k> ≈ 8-10
- **拟合误差：** < 5%

## 3. phase_transition.png - 相变图（包含逾渗相变）⭐

### 📐 图表内容

**2×2子图布局：**

**左上：采纳率相变**
- **X轴：** 补贴强度（元）
- **Y轴：** 采纳率（0-1）
- **蓝点：** 仿真数据
- **红曲线：** Sigmoid拟合曲线
- **垂直虚线：** 临界补贴点（采纳率突变点）

**右上：逾渗相变（巨大连通分量）⭐**
- **X轴：** 补贴强度（元）
- **Y轴：** 巨大连通分量大小（0-1）
- **蓝点：** 仿真数据
- **红曲线：** 拟合曲线
- **垂直虚线：** 逾渗临界点（巨大连通分量出现点）
- **区域标注：** 亚临界区、临界区、超临界区

**左下：序参数关系**
- **X轴：** 采纳率
- **Y轴：** 巨大连通分量大小
- **散点：** 数据点
- **拟合曲线：** 二次函数拟合

**右下：三个序参数对比**
- **X轴：** 补贴强度（元）
- **Y轴：** 归一化值（0-1）
- **三条曲线：**
  - 紫色：采纳率
  - 蓝色：巨大连通分量
  - 橙色：全局质量

### 🎯 科学意义

1. **相变现象：**
   - 当补贴达到**临界值**时，采纳率发生**突然跳跃**
   - 从低采纳率（< 30%）突然跳到高采纳率（> 70%）
   - 这是**集体行为的涌现**

2. **逾渗相变（⭐创新）：**
   - 巨大连通分量突然出现
   - 表示采纳者从**分散的小簇**变成**连成一片的大簇**
   - 体现了"集体行动"的涌现

3. **临界点识别：**
   - **临界补贴：** 约1500-2000元
   - **临界采纳率：** 约30-40%
   - **逾渗临界点：** 巨大连通分量出现的补贴值

4. **序参数关系：**
   - 三个序参数（采纳率、巨大连通分量、全局质量）的**协同演化**
   - 展示系统从无序到有序的转变

### 📊 关键信息

- **临界补贴：** s_c ≈ 1500-2000元
- **临界采纳率：** ρ_c ≈ 30-40%
- **逾渗临界点：** s_p ≈ 1800元
- **巨大连通分量：** 在临界点突然出现

## 4. betweenness_response.png - 中心性分析图

### 📐 图表内容

**散点图 + 拟合曲线**

- **X轴：** 中介中心性（Betweenness Centrality）
  - 衡量农户在网络中的"桥梁"作用
  - 值越大，越多的信息流经过该农户

- **Y轴：** 政策响应度（Policy Response）
  - 综合指标：采纳状态 + 政策感知度 + 社会影响
  - 值越大，对政策越敏感

- **散点：** 每个点代表一个农户
- **拟合曲线：** 线性或非线性拟合
- **颜色：** 可能按采纳状态着色

### 🎯 科学意义

1. **网络位置与政策响应：**
   - 验证**中心性假设**：网络中心位置越高的农户，对政策越敏感
   - Hub节点（高中心性）更容易采纳

2. **政策传播路径：**
   - 高中心性农户是**关键传播节点**
   - 影响这些农户可以加速政策扩散

3. **异质性分析：**
   - 展示不同网络位置的农户响应差异
   - 为**精准施策**提供依据

### 📊 关键信息

- **响应度标准差：** 衡量异质性
- **拟合斜率：** 中心性对响应的影响强度
- **R²值：** 拟合优度

## 5. diffusion_time.png - 扩散动力学图

### 📐 图表内容

**时间序列图**

- **X轴：** 时间步（0-60）
- **Y轴：** 采纳率（0-1）
- **曲线：** 采纳率随时间的变化
- **可能包含：**
  - 实际扩散曲线
  - 理论预测曲线
  - 不同初始条件的对比

### 🎯 科学意义

1. **扩散速度：**
   - 展示采纳行为在**网络中的传播速度**
   - 从初始5%到最终采纳率的演化过程

2. **扩散模式：**
   - **S型曲线：** 初期慢 → 中期快 → 后期慢
   - 符合**创新扩散理论**（Rogers, 1962）

3. **网络效应：**
   - 对比有网络和无网络的扩散速度
   - 展示**社会网络加速扩散**的效应

4. **关键时间点：**
   - **加速点：** 扩散速度最快的时刻
   - **饱和点：** 扩散速度减慢的时刻

### 📊 关键信息

- **扩散速度系数：** 衡量传播快慢
- **加速时间点：** 扩散最快的时刻
- **最终采纳率：** 稳态值
