## 1. 临界点的判断方法

### 方法1：梯度最大法（采纳率相变）

#### 📊 对应图表：左图 "Adoption Rate Phase Transition"

#### 🔍 判断原理

**步骤：**

1. **计算梯度（变化率）**
   ```python
   gradient_adoption = np.gradient(adoption_rates)
   # 计算采纳率随补贴的变化率
   ```

2. **找到梯度最大的点**
   ```python
   critical_idx = np.argmax(gradient_adoption)
   # 找到变化率最大的补贴点
   ```

3. **确定临界补贴和临界阈值**
   ```python
   critical_subsidy = subsidy_range[critical_idx]
   critical_threshold = adoption_rates[critical_idx]
   ```

#### 📈 图示说明

```
补贴（元）    采纳率    梯度（变化率）
500           0.10      -
632           0.15      0.05/132 = 0.00038
763           0.20      0.05/131 = 0.00038
895           0.25      0.05/132 = 0.00038
...
1816          0.73      0.48/921 = 0.00052  ⭐ 梯度最大！
1947          0.91      0.18/131 = 0.00137
2079          0.89      -0.02/132 = -0.00015
```

**关键点：**
- 梯度最大的点 = **变化最快**的点
- 在这个点，小幅增加补贴，采纳率**大幅提升**
- 这就是**临界点**

#### 🎯 在图中如何识别

**左图 "Adoption Rate Phase Transition"：**

1. **绿线（Critical Subsidy = 1816）**
   - 垂直虚线标记临界补贴
   - 这是梯度最大的补贴点

2. **橙线（Critical Threshold = 0.73）**
   - 水平虚线标记临界采纳率
   - 这是临界点对应的采纳率

3. **S型曲线**
   - 红色虚线是Sigmoid拟合曲线
   - 临界点在S型曲线的**中间拐点**（最陡的地方）

4. **判断依据：**
   - 临界点 = S型曲线的**拐点**（最陡的地方）
   - 临界点 = 梯度最大的点
   - 临界点 = 采纳率从慢速增长到快速增长的转折点

## 3. 两个临界点的关系

### 🎯 相同点

1. **临界补贴相同：1816元**
   - 采纳率相变的临界点：1816元
   - 逾渗相变的临界点：1816元

2. **临界阈值相同：0.73（73%）**
   - 采纳率相变的临界阈值：0.73
   - 逾渗相变的临界阈值：0.73

3. **发生时间相同**
   - 两个相变**同时发生**
   - 说明它们是**相关的**

---

### 🔗 因果关系

**解释：**

1. **采纳率提高** → **巨大连通分量出现**
   - 当更多农户采纳时，采纳者之间的连接增加
   - 当连接足够多时，巨大连通分量突然出现

2. **巨大连通分量出现** → **采纳率进一步提高**
   - 巨大连通分量形成后，内部农户相互影响
   - 加速了采纳率的进一步提高

3. **正反馈机制**
   - 采纳率 ↑ → 巨大连通分量 ↑ → 采纳率 ↑↑
   - 形成**正反馈循环**，导致相变

## 4. 判断标准的量化指标

### 📊 临界点判断标准

#### 标准1：梯度最大

```python
# 计算梯度
gradient = np.gradient(y_values)

# 找到梯度最大的点
critical_idx = np.argmax(gradient)

# 判断：梯度是否足够大
if gradient[critical_idx] > threshold:
    # 这是临界点
    critical_point = x_values[critical_idx]
```

**阈值：**
- 梯度 > 0.0005：认为是临界点
- 梯度 < 0.0005：认为是平滑过渡

---

#### 标准2：S型曲线拐点

```python
# 拟合Sigmoid曲线
def sigmoid(x, L, k, x0):
    return L / (1 + np.exp(-k * (x - x0)))

# 拟合参数
L, k, x0 = curve_fit(sigmoid, x, y)

# 拐点 = x0（Sigmoid曲线的中心点）
critical_point = x0
```

**判断：**
- 拐点（x0）就是临界点
- R² > 0.7：拟合良好，可以确定临界点

---

#### 标准3：巨大连通分量突然出现

```python
# 计算巨大连通分量
giant_sizes = [compute_giant_component_size(...) for subsidy in subsidy_range]

# 找到突然跳跃的点
for i in range(1, len(giant_sizes)):
    if giant_sizes[i] - giant_sizes[i-1] > 0.3:  # 跳跃 > 30%
        percolation_point = subsidy_range[i]
        break
```

**判断：**
- 跳跃 > 30%：认为是逾渗相变
- 跳跃 < 30%：认为是平滑过渡

## 7. 总结

### 🎯 临界点判断方法

1. **梯度最大法**
   - 计算梯度（变化率）
   - 找到梯度最大的点
   - 这就是临界点

2. **S型曲线拐点法**
   - 拟合Sigmoid曲线
   - 拐点（x0）就是临界点

3. **突然跳跃法**
   - 观察巨大连通分量的突然跳跃
   - 跳跃点就是逾渗临界点

---

   - 突然跳跃（> 30%）
