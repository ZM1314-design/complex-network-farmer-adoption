## 三、强化学习状态向量（12维）

### 3.1 状态向量定义

**状态空间维度：** `state_dim = 12`

**状态向量组成：**

$$
s_i(t) = \left[s_1, s_2, s_3, s_4, s_5, s_6, s_7, s_8, s_9, s_{10}, s_{11}, s_{12}\right]
$$

### 3.2 各维度详细说明

| 维度 | 符号 | 名称 | 范围 | 计算方式 | 说明 |
|------|------|------|------|----------|------|
| **s₁** | `income_i` | 经济水平 | [0, 1] | `economic_level / 100000` | 归一化的经济水平 |
| **s₂** | `social_i` | 社会影响力权重 | [0, 1] | `degree / max_degree` | 基于度中心性归一化 |
| **s₃** | `policy_i` | 政策感知 | [0, 1] | `policy_perception` | 农户对补贴的感知度 |
| **s₄** | `risk_i` | 风险偏好 | [0, 1] | `risk_tolerance` | 风险承受能力 |
| **s₅** | `F_i` | 肥力 | [0, 1] | `fertility` | 当前肥力值 |
| **s₆** | `S_i` | 颗粒结构 | [0, 1] | `soil_structure` | 当前颗粒结构值 |
| **s₇** | `B_i` | 生物活性 | [0, 1] | `biological_activity` | 当前生物活性值 |
| **s₈** | `ρ_Ni(t)` | 邻居采纳率 | [0, 1] | `mean(neighbors.adoption_state)` | 邻居中采纳绿色施肥的比例 |
| **s₉** | `Q(t)` | 全局质量 | [0, 1] | `mean(all_farmers.land_quality)` | 系统平均耕地质量 |
| **s₁₀** | `δ_i` | 短视性 | [0.1, 0.9] | `myopia` | 短视性参数，影响折扣因子 |
| **s₁₁** | `E_i` | 子女教育优先权 | [0, 1] | `education_priority` | 当前教育优先权水平 |
| **s₁₂** | `V_i` | 农民话语权 | [0, 1] | `political_voice` | 当前话语权（参政机会）水平 |

### 3.3 状态向量代码实现

```python
state = np.array([
    self.attrs['economic_level'] / 100000,  # s₁: 归一化经济水平
    social_influence,                        # s₂: 社会影响力权重
    self.attrs['policy_perception'],         # s₃: 政策感知
    self.attrs['risk_tolerance'],           # s₄: 风险偏好
    fertility,                               # s₅: 肥力
    soil_structure,                          # s₆: 颗粒结构
    biological_activity,                     # s₇: 生物活性
    neighbor_adoption,                       # s₈: 邻居采纳率
    global_Q,                                # s₉: 全局质量
    myopia,                                  # s₁₀: 短视性
    education_priority,                      # s₁₁: 子女教育优先权
    political_voice                          # s₁₂: 农民话语权
], dtype=np.float32)
```

### 3.4 动作空间

**动作空间：** `action_dim = 2`

| 动作 | 值 | 说明 |
|------|-----|------|
| 传统施肥 | 0 | 不采纳绿色施肥 |
| 绿色施肥 | 1 | 采纳绿色施肥 |

### 3.5 DQN网络架构

```
输入层：  12维状态向量
    ↓
隐藏层1： 64维（ReLU激活）
    ↓
隐藏层2： 64维（ReLU激活）
    ↓
输出层：  2维（Q值，对应两个动作）
```

**网络参数：**
- 输入维度：12
- 隐藏层维度：64
- 输出维度：2
- 激活函数：ReLU
- 优化器：Adam (lr=0.001)

## 四、数据流总结

### 4.1 数据生成流程

1. **基础属性生成** → `economic_level`, `education`, `risk_tolerance`, `env_awareness`
2. **短视性生成** → `myopia` → 计算 `discount_factor`
3. **新维度生成** → `education_priority`, `political_voice`（基于教育、经济水平）
4. **质量生成** → `fertility`, `soil_structure`, `biological_activity` → 计算 `land_quality`
5. **位置生成** → `x`, `y`, `z`（用于网络构建）
6. **政策属性生成** → `policy_perception`, `training_access`

### 4.2 状态向量构建流程

1. **从DataFrame获取** → 基础属性、质量、新维度
2. **从网络计算** → 社会影响力权重（度中心性）
3. **从邻居计算** → 邻居采纳率
4. **从系统获取** → 全局质量
5. **归一化处理** → 所有值归一化到[0,1]或指定范围
6. **组装状态向量** → 12维numpy数组

### 4.3 参数使用流程

1. **网络构建** → 使用网络参数构建双曲几何网络
2. **质量演化** → 使用质量动力学参数更新质量
3. **决策计算** → 使用TPB决策参数计算效用
4. **RL训练** → 使用RL参数训练DQN网络
5. **奖励计算** → 使用奖励权重计算总奖励

## 五、关键公式

### 5.1 状态向量公式

$$
s_i(t) = \left[\text{income}_i, \text{social}_i, \text{policy}_i, \text{risk}_i, F_i, S_i, B_i, \rho_{\mathcal{N}_i}(t), Q(t), \delta_i, E_i, V_i\right]
$$

### 5.2 折扣因子公式

$$
\gamma_i = \frac{1}{1 + \delta_i}
$$

### 5.3 综合质量公式

$$
Q_i(t) = 0.4 \cdot F_i(t) + 0.3 \cdot S_i(t) + 0.3 \cdot B_i(t)
$$

### 5.4 奖励函数公式

$$
R_i(S, A) = \lambda_1 \cdot R_{\text{economic}} + \lambda_2 \cdot R_{\text{policy}} + \lambda_3 \cdot R_{\text{social}} + \lambda_4 \cdot R_{\text{education}} + \lambda_5 \cdot R_{\text{voice}}
$$
