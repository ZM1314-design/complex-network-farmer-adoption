## 📊 将生成的所有图表

### 核心图表（8张）

1. **`network_final_state.png`** - 网络结构图
   - 3D可视化网络结构
   - 节点颜色表示采纳状态（绿色=采纳，灰色=传统）

2. **`degree_distribution.png`** - 度分布图
   - 直方图和对数-对数图
   - 验证幂律分布 P(k) ~ k^(-2.5)

3. **`phase_transition.png`** - 相变图（包含逾渗相变）⭐
   - 2x2子图布局
   - 采纳率相变、逾渗相变（巨大连通分量）、序参数关系、三个序参数对比

4. **`betweenness_response.png`** - 中心性分析图
   - 中介中心性与政策响应度的关系
   - 散点图 + 拟合曲线

5. **`diffusion_time.png`** - 扩散动力学图
   - 采纳率随时间的变化
   - 展示网络传播效应

6. **`quality_3d_evolution.png`** - 三维质量演化图 ⭐
   - 2x2子图布局
   - 三个维度（肥力、颗粒结构、生物活性）的演化曲线

7. **`training_curves.png`** - RL训练曲线
   - 采纳率、平均奖励、全局质量、探索率随训练轮数的变化

8. **`comparison.png`** - 基线 vs RL 对比图
   - 采纳率对比、质量对比

## 🔧 运行参数

### 配置文件
默认使用 `config.yaml`，可以通过参数指定：

```bash
python run_complete_experiment.py --config config.yaml
```

### 主要参数（在 `config.yaml` 中）

```yaml
network:
  num_farmers: 300  # 农户数量

simulation:
  num_episodes: 50  # RL训练轮数
  max_steps: 60     # 每轮最大步数

policy:
  subsidy_base: 1000  # 基础补贴
  subsidy_range: [500, 3000]  # 补贴范围
```

## 📝 运行步骤详解

### 步骤1：基线仿真
1. 生成农户数据（300个农户，18个字段）
2. 构建复杂网络（双曲几何，无标度网络）
3. 运行动力学仿真（60个时间步）
4. 相变分析（扫描补贴范围，分析逾渗相变）
5. 生成基线图表（6张）

### 步骤2：强化学习训练
1. 准备环境（数据、网络）
2. 初始化多智能体系统（300个DQN智能体）
3. 强化学习训练（50个episode）
4. 生成训练曲线图

### 步骤3：对比分析
1. 加载基线结果和RL结果
2. 生成对比图表

### 步骤4：生成补充图表
1. 政策工具效果对比
2. 政策协同效应
3. 干预时机分析
4. 退出策略分析
5. 群体异质性热力图
6. 从众效应分析
7. 年龄组响应分析
