## 📖 项目概述

### 核心功能

本项目实现了一个完整的复杂网络动力学模型，用于研究农户绿色施肥技术采纳行为：

#### ✅ 已实现的模型

1. **双曲几何无标度网络**
   - 度分布：P(k) ~ k^(-γ)
   - 连边概率：P(i↔j) = exp(-ζ·d_ij)
   - 超边支持（合作社）
   - 村内强连接

2. **耕地质量动力学**
   - dQ/dt = α·ρ_g - β·ρ_t - γ·Q + δ
   - 个体-集体双层演化
   - 邻居扩散效应

3. **TPB决策理论**
   - 效用函数：U_i = θ_p·S_policy + θ_s·S_social - θ_e·C_cost + θ_q·Q(t)
   - Logit选择模型
   - 临界阈值判据

4. **相变分析**
   - 临界阈值检测
   - Sigmoid曲线拟合
   - 网络传播效应
   - 超边修正

5. **DQN强化学习**
   - 多智能体系统
   - 个性化折扣因子
   - 四项奖励函数
   - ε-greedy探索

#### 📊 输出结果

- 7张高质量可视化图表
- 完整的时间序列数据
- 关键指标摘要
- 训练好的模型

## ❓ 常见问题

### Q1: 运行时报错 "ModuleNotFoundError"

**原因：** 依赖包未安装

**解决：**
```bash
pip install -r requirements.txt
```

### Q2: 训练太慢

**解决：** 使用当前的平衡版配置，或进一步减少：
- 农户数量：200
- 训练轮数：20-30

### Q3: 图表中文乱码

**解决：** 正常现象，不影响使用。或在 `visualization.py` 中改用英文字体。

### Q4: 内存不足

**解决：**
- 减少农户数量
- 减小 `memory_size: 3000`
- 减小 `batch_size: 64`

### Q5: 想看详细错误信息

**解决：** 在命令行运行，可以看到完整堆栈：
```bash
python run_experiment.py --mode all
```

### Q6: 如何暂停和继续？

**暂停：** 按 `Ctrl+C`

**继续：** 模型会自动保存检查点（如果实现了），或重新运行。
