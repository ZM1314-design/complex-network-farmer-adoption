## 🔧 常见问题

### Q1: 找不到文件或目录
**错误信息：** `The system cannot find the path specified`

**原因：** 没有进入项目目录

**解决方案：**
```bash
# 1. 确认当前位置
pwd  # PowerShell
cd   # CMD

# 2. 进入正确目录
cd C:\Users\zhaom\Desktop\复杂网络

# 3. 验证（应该能看到 run_experiment.py）
dir
```

---

### Q2: 路径中有中文导致问题
**错误信息：** 路径乱码或找不到

**解决方案：**
```bash
# 使用引号包裹路径
cd "C:\Users\zhaom\Desktop\复杂网络"

# 或者使用短路径名（8.3格式）
cd C:\Users\zhaom\Desktop\FUZA~1
```

---

### Q3: 不确定是否在正确目录
**验证方法：**

```bash
# 查看当前目录
pwd  # PowerShell
cd   # CMD

# 列出文件（应该看到这些文件）
dir

# 应该看到：
# - run_experiment.py
# - config.yaml
# - requirements.txt
# - src/ 文件夹
# - figures/ 文件夹
```

---

### Q4: 每次都要重新cd吗？
**答案：** 是的，每次打开新的命令行窗口都需要cd

**解决方案：**
- **方法A：** 使用批处理文件（`一键运行完整实验.bat`），自动cd
- **方法B：** 在项目文件夹中右键 → "在终端中打开"
- **方法C：** 在VS Code中打开项目，使用集成终端（自动在正确目录）

## 💡 专业技巧

### 技巧1：在文件夹中直接打开终端

**Windows 11/10：**
1. 打开文件资源管理器
2. 导航到项目文件夹（`C:\Users\zhaom\Desktop\复杂网络`）
3. 在地址栏输入 `cmd` 或 `powershell` 并回车
4. 自动在正确目录打开终端！

---

### 技巧2：使用VS Code

1. 用VS Code打开项目文件夹
2. 按 `` Ctrl + ` `` 打开集成终端
3. 终端自动在项目目录下
4. 直接运行命令：
   ```bash
   python run_experiment.py
   ```

---

### 技巧3：创建桌面快捷方式

**创建批处理文件：** `运行实验.bat`

```batch
@echo off
cd /d "C:\Users\zhaom\Desktop\复杂网络"
python run_experiment.py
pause
```

**使用方法：**
1. 将此文件保存到桌面
2. 双击即可运行
3. 无需手动cd
