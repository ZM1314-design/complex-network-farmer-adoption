## 📐 完整数学公式体系

### 一、网络构建公式（Network Construction）

#### 1.1 双曲几何距离

**双曲空间中两节点间的距离：**

\[
d_{ij} = \text{arcosh}\left(1 + \frac{2\|x_i - x_j\|^2}{(1-\|x_i\|^2)(1-\|x_j\|^2)}\right)
\]

其中：
- \(x_i, x_j \in \mathbb{R}^3\)：节点在双曲空间的3D坐标
- \(\|x_i\| < 1\)：坐标需在庞加莱球内

#### 1.2 连边概率函数

**基于距离的连边概率：**

\[
P(i \leftrightarrow j) = \exp(-\zeta \cdot d_{ij})
\]

参数：
- \(\zeta \approx 2.5\)：距离敏感性参数
- 距离越远，连边概率越低

#### 1.3 度分布幂律

**无标度网络的度分布：**

\[
P(k) \sim k^{-\gamma}
\]

其中：
- \(\gamma \approx 2.5\)：幂律指数
- \(k\)：节点度数

**网络异质性指标：**

\[
h = \frac{\sigma_k}{\langle k \rangle} = \frac{\sqrt{\langle k^2 \rangle - \langle k \rangle^2}}{\langle k \rangle}
\]

目标：\(h \approx 0.46\)

#### 1.4 超边密度参数

**超边比例计算：**

\[
\delta = \frac{3\rho}{2+\rho}
\]

其中：
- \(\rho\)：合作社参与率
- \(\delta\)：超边在总边中的比例

### 二、动力学演化公式（Dynamics）

#### 2.1 全局耕地质量演化

**宏观质量微分方程：**

\[
\frac{dQ}{dt} = \alpha \rho_g - \beta \rho_t - \gamma Q + \delta
\]

参数定义：
- \(Q(t)\)：全局耕地质量指数 \([0,1]\)
- \(\rho_g(t) = \frac{1}{N}\sum_{i=1}^N a_i\)：绿色施肥比例（序参数）
- \(\rho_t(t) = 1 - \rho_g\)：传统施肥比例
- \(\alpha = 0.05\)：绿色施肥质量提升系数
- \(\beta = 0.08\)：传统施肥质量退化系数
- \(\gamma = 0.02\)：自然退化率
- \(\delta = 0.01\)：外部生态修复项

**物理意义：**
- \(\alpha \rho_g\)：集体绿色施肥的正反馈
- \(-\beta \rho_t\)：传统施肥的负面影响
- \(-\gamma Q\)：质量自然衰减（阻尼）
- \(+\delta\)：外部修复（驱动力）

#### 2.2 个体质量演化（扩散项）

**个体质量微分方程：**

\[
\frac{dQ_i}{dt} = f(a_i) - \gamma Q_i + \delta + D \nabla^2 Q_i
\]

其中：

\[
f(a_i) = \begin{cases}
\alpha & \text{if } a_i = 1 \text{ (绿色施肥)} \\
-\beta & \text{if } a_i = 0 \text{ (传统施肥)}
\end{cases}
\]

**网络拉普拉斯算子（扩散项）：**

\[
\nabla^2 Q_i \approx \frac{\sum_{j \in \mathcal{N}_i} w_{ij}(Q_j - Q_i)}{\sum_{j \in \mathcal{N}_i} w_{ij}}
\]

- \(\mathcal{N}_i\)：节点\(i\)的邻居集合
- \(w_{ij}\)：边权重
- \(D\)：扩散系数

### 三、决策理论公式（Decision Theory）

#### 3.1 TPB效用函数

**综合效用函数：**

\[
U_i = \theta_p \cdot S_{\text{policy}} + \theta_s \cdot S_{\text{social}} - \theta_e \cdot C_{\text{cost}} + \theta_q \cdot Q_i(t)
\]

**各项详细公式：**

##### 3.1.1 政策激励项

\[
S_{\text{policy}} = \frac{\text{subsidy}}{\text{cost}_{\text{green}}} \times \text{perception}_i \times (1 + \text{training}_i \times 0.2)
\]

##### 3.1.2 社会网络影响项

**普通农户：**

\[
S_{\text{social}} = \frac{\sum_{j \in \mathcal{N}_i} w_{ij} \cdot a_j}{\sum_{j \in \mathcal{N}_i} w_{ij}}
\]

**合作社成员（超边）：**

\[
S_{\text{social}} = 0.7 \times S_{\text{neighbor}} + 0.3 \times S_{\text{coop}}
\]

其中：
- \(S_{\text{neighbor}}\)：普通邻居采纳率
- \(S_{\text{coop}}\)：合作社内采纳率

##### 3.1.3 成本项

\[
C_{\text{cost}} = \begin{cases}
\text{cost}_{\text{green}} - \text{subsidy} & \text{if } a_i = 1 \\
\text{cost}_{\text{traditional}} & \text{if } a_i = 0
\end{cases}
\]

##### 3.1.4 质量项

\[
Q_i(t) \in [0, 1]
\]

#### 3.2 质量-政策耦合（⭐创新）

**政策感知权重的非线性调制：**

\[
\theta_p = \theta_{p0} \cdot \tanh(\mu \cdot Q)
\]

参数：
- \(\theta_{p0}\)：基础政策权重
- \(\mu\)：耦合强度参数
- \(\tanh\)：双曲正切函数

**物理意义：**
- \(Q \to 0\)：质量差 → \(\tanh \to 0\) → 对政策不敏感
- \(Q \to \infty\)：质量好 → \(\tanh \to 1\) → 政策敏感度饱和

#### 3.3 Logit选择模型

**采纳概率：**

\[
P(a_i = 1) = \frac{1}{1 + \exp(-\tau \cdot U_i + \epsilon_i)}
\]

其中：
- \(\tau\)：温度参数（理性程度）
- \(\epsilon_i \sim \text{Gumbel}(0, 1)\)：随机扰动项

**等价形式（Logistic函数）：**

\[
P(a_i = 1) = \frac{\exp(\tau \cdot U_i)}{\exp(\tau \cdot U_i) + \exp(0)} = \sigma(\tau \cdot U_i)
\]

### 四、相变分析公式（Phase Transition）

#### 4.1 临界阈值判据

**个体采纳条件（阈值模型）：**

\[
\frac{\sum_{j \in \mathcal{N}_i} w_{ij} \cdot a_j}{\max_k w_{ik}} + \frac{P_i}{P_{\max}} - \frac{C_i}{C_{\max}} \geq \Theta_c
\]

三项平衡：
- 第一项：归一化社会影响
- 第二项：归一化政策激励
- 第三项：归一化成本阻力
- \(\Theta_c\)：临界阈值

#### 4.2 超边修正的临界阈值（⭐创新）

**考虑超边后的阈值修正：**

\[
\Theta_c^* = \Theta_c \cdot (1 - p_h)^{-\kappa}
\]

其中：

\[
\kappa = \frac{\ln(\langle k_{\text{hyper}} \rangle / \langle k \rangle)}{\ln(1 + p_h)}
\]

参数：
- \(p_h\)：超边比例
- \(\langle k_{\text{hyper}} \rangle\)：超边增强后的平均度
- \(\langle k \rangle\)：原始平均度
- \(\kappa\)：修正指数

**物理意义：**
- 超边增加 → \(p_h\) 增大 → \(\Theta_c^*\) 减小
- 合作社降低了采纳门槛

#### 4.3 Sigmoid相变曲线

**采纳率随补贴的函数关系：**

\[
\rho(s) = \frac{L}{1 + \exp[-k(s - s_0)]}
\]

参数：
- \(s\)：补贴强度
- \(L\)：饱和采纳率（上限）
- \(k\)：陡峭程度（相变强度）
- \(s_0\)：临界补贴强度（拐点位置）

**临界点判据（一阶导数最大）：**

\[
s_c = \arg\max_s \frac{d\rho}{ds}
\]

**拟合优度：**

\[
R^2 = 1 - \frac{\sum_i (\rho_i - \hat{\rho}_i)^2}{\sum_i (\rho_i - \bar{\rho})^2}
\]

#### 4.4 网络扩散速度理论

**理论扩散速度：**

\[
v \propto \sqrt{\gamma \cdot k_{\max}}
\]

其中：
- \(\gamma\)：度分布指数
- \(k_{\max}\)：网络最大度数

**Hub节点杠杆效应：**

\[
\text{Leverage} = \frac{\Delta \rho_{\text{hub}}}{\Delta \rho_{\text{random}}}
\]

- \(\Delta \rho_{\text{hub}}\)：优先激励Hub节点的采纳率提升
- \(\Delta \rho_{\text{random}}\)：随机激励的采纳率提升

### 五、强化学习公式（Reinforcement Learning）

#### 5.1 状态空间

**多维状态向量：**

\[
\mathbf{S}_i = \begin{bmatrix}
s_1 \\ s_2 \\ s_3 \\ s_4 \\ s_5 \\ s_6 \\ s_7
\end{bmatrix} = \begin{bmatrix}
\text{经济水平}/100000 \\
\text{度中心性} \\
\text{政策感知度} \\
\text{风险偏好} \\
Q_i(t) \\
\sum_{j} w_{ij} a_j / \sum_j w_{ij} \\
Q(t)
\end{bmatrix}
\]

所有分量归一化到 \([0, 1]\)

#### 5.2 Q-Learning核心公式

**Bellman方程：**

\[
Q(s, a) = \mathbb{E}\left[R_t + \gamma \max_{a'} Q(s', a') \mid s_t = s, a_t = a\right]
\]

**DQN更新规则：**

\[
\theta \leftarrow \theta + \alpha \left[R + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta)\right] \nabla_\theta Q(s, a; \theta)
\]

参数：
- \(\theta\)：Q网络参数
- \(\theta^-\)：目标网络参数
- \(\alpha\)：学习率
- \(\gamma\)：折扣因子

#### 5.3 个性化折扣因子（⭐创新）

**短视性调制的折扣因子：**

\[
\gamma_i = \frac{1}{1 + \delta_i}
\]

其中：
- \(\delta_i \sim \mathcal{U}(0.1, 0.9)\)：农户短视性参数
- \(\delta_i\) 大 → \(\gamma_i\) 小 → 短视（只看眼前）
- \(\delta_i\) 小 → \(\gamma_i\) 大 → 远见（考虑长期）

#### 5.4 奖励函数

**四维加权奖励：**

\[
R_i = \lambda_1 R_{\text{economic}} + \lambda_2 R_{\text{subsidy}} + \lambda_3 R_{\text{social}} + \lambda_4 R_{\text{education}}
\]

**各项详细公式：**

##### 5.4.1 经济效益

\[
R_{\text{economic}} = \begin{cases}
Y \times (1 + 0.5 Q_i) - C_g + S & \text{if } a_i = 1 \\
Y \times (1 + 0.2 Q_i) - C_t & \text{if } a_i = 0
\end{cases}
\]

- \(Y\)：基础产量
- \(C_g\)：绿色施肥成本
- \(C_t\)：传统施肥成本
- \(S\)：补贴

##### 5.4.2 政策补贴

\[
R_{\text{subsidy}} = \begin{cases}
S & \text{if } a_i = 1 \\
0 & \text{if } a_i = 0
\end{cases}
\]

##### 5.4.3 社会声誉（从众奖励）

\[
R_{\text{social}} = \begin{cases}
+\phi \times \bar{a}_{\mathcal{N}_i} & \text{if } a_i = \bar{a}_{\mathcal{N}_i} \\
-\phi \times (1 - \bar{a}_{\mathcal{N}_i}) & \text{if } a_i \neq \bar{a}_{\mathcal{N}_i}
\end{cases}
\]

- \(\bar{a}_{\mathcal{N}_i}\)：邻居平均采纳率
- \(\phi\)：从众压力强度

##### 5.4.4 教育优先权

\[
R_{\text{education}} = \begin{cases}
\psi & \text{if } a_i = 1 \text{ and has children} \\
0 & \text{otherwise}
\end{cases}
\]

#### 5.5 ε-greedy探索策略

**探索率衰减：**

\[
\epsilon(t) = \max(\epsilon_{\min}, \epsilon_0 \times \epsilon_{\text{decay}}^t)
\]

参数：
- \(\epsilon_0 = 1.0\)：初始探索率
- \(\epsilon_{\text{decay}} = 0.99\)：衰减率
- \(\epsilon_{\min} = 0.05\)：最小探索率

**动作选择：**

\[
a_t = \begin{cases}
\text{random}(\{0, 1\}) & \text{with probability } \epsilon(t) \\
\arg\max_a Q(s_t, a; \theta) & \text{with probability } 1 - \epsilon(t)
\end{cases}
\]

### 六、网络中心性指标

#### 6.1 度中心性

\[
C_D(i) = \frac{k_i}{N-1}
\]

#### 6.2 中介中心性

\[
C_B(i) = \sum_{s \neq i \neq t} \frac{\sigma_{st}(i)}{\sigma_{st}}
\]

- \(\sigma_{st}\)：节点\(s\)到\(t\)的最短路径数
- \(\sigma_{st}(i)\)：经过节点\(i\)的最短路径数

#### 6.3 特征向量中心性

\[
\mathbf{Ax} = \lambda \mathbf{x}
\]

- \(\mathbf{A}\)：邻接矩阵
- \(\lambda\)：最大特征值
- \(\mathbf{x}\)：对应的特征向量

---

### 七、统计指标

#### 7.1 聚类系数

**节点\(i\)的局部聚类系数：**

\[
C_i = \frac{2 e_i}{k_i(k_i - 1)}
\]

- \(e_i\)：节点\(i\)邻居间的实际连边数
- \(k_i\)：节点度数

**全局聚类系数：**

\[
C = \frac{1}{N} \sum_{i=1}^N C_i
\]

#### 7.2 平均路径长度

\[
L = \frac{1}{N(N-1)} \sum_{i \neq j} d_{ij}
\]

- \(d_{ij}\)：节点\(i\)到\(j\)的最短路径长度

#### 7.3 网络异质性

\[
h = \frac{\sigma_k}{\langle k \rangle}
\]

- \(\sigma_k = \sqrt{\text{Var}(k)}\)：度数标准差
- \(\langle k \rangle = \frac{1}{N}\sum_i k_i\)：平均度数

### 八、损失函数与优化

#### 8.1 DQN损失函数

**时序差分误差：**

\[
\mathcal{L}(\theta) = \mathbb{E}_{(s,a,r,s') \sim \mathcal{D}}\left[\left(r + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta)\right)^2\right]
\]

- \(\mathcal{D}\)：经验回放池
- \(\theta^-\)：目标网络参数（定期从\(\theta\)更新）

#### 8.2 优化器

**Adam优化：**

\[
\theta_{t+1} = \theta_t - \frac{\alpha}{\sqrt{\hat{v}_t} + \epsilon} \hat{m}_t
\]

其中：
- \(\hat{m}_t\)：一阶矩估计（动量）
- \(\hat{v}_t\)：二阶矩估计（自适应学习率）
- \(\alpha = 0.001\)：学习率

```python
β = 0.08  # 传统施肥退化系数
γ = 0.02  # 自然退化率
δ = 0.01  # 外部生态修复作用

ρ_g = Σa_i/N  # 绿色施肥比例（序参数p(t)）
ρ_t = 1 - ρ_g # 传统施肥比例

【物理意义】
- α·ρ_g：集体采纳绿色施肥的正反馈
- β·ρ_t：传统施肥的负面影响
- γ·Q：质量自然衰减
- δ：政府生态治理的基础作用

【创新点4】序参数-质量耦合
- Q(t)既是状态变量，又影响决策
- 形成"质量提升 → 更多采纳 → 质量进一步提升"的正反馈
- 或"质量下降 → 放弃绿色 → 质量进一步下降"的负反馈
```

#### 2.2 个体质量演化（扩散项）

```python
【个体动力学】
dQ_i/dt = {
    α - γ·Q_i + δ + D·∇²Q_i    if a_i = 1 (绿色)
    -β - γ·Q_i + δ + D·∇²Q_i   if a_i = 0 (传统)
}

【扩散项】
∇²Q_i ≈ Σ_j w_ij(Q_j - Q_i) / Σ_j w_ij

邻居质量会通过网络传播影响个体质量

【创新点5】个体-集体双层动力学
- 全局Q(t)：宏观序参数
- 局部Q_i(t)：微观质量
- 通过扩散项耦合，形成多尺度演化
```

#### 2.3 TPB决策理论

```python
【效用函数】
U_i = θ_p·S_policy + θ_s·S_social - θ_e·C_cost + θ_q·Q_i(t)

【各项详解】
1. 政策激励项：
   S_policy = (subsidy/cost_green) × perception_i × (1+training_i×0.2)
   
2. 社会网络项：
   S_social = Σ_j w_ij·a_j / Σ_j w_ij
   如果在合作社中：
   S_social = 0.7×邻居采纳率 + 0.3×合作社采纳率
   
3. 成本项：
   C_cost = {
       cost_green - subsidy  if a_i = 1
       cost_traditional      if a_i = 0
   }
   
4. 质量项：
   直接使用Q_i(t)

【权重耦合机制】⭐ 创新点6
θ_p = θ_p0 · tanh(μ·Q)

物理意义：
- Q → 0：质量差，对政策不敏感（tanh→0）
- Q → ∞：质量好，政策敏感度饱和（tanh→1）
- 非线性耦合，体现质量对政策感知的调制作用
```

#### 2.4 Logit选择模型

```python
【概率决策】
P(a_i = 1) = 1 / (1 + exp(-τ·U_i + ε_i))

τ：温度参数（理性程度）
ε_i ~ Gumbel(0,1)：随机误差

【与阈值判据的对比】
Logit模型：渐变式采纳（概率上升）
阈值模型：突变式采纳（0/1跳变）

两者结合，既有平滑演化又有相变现象
```

### 阶段三：相变分析与临界阈值（Phase Transition）

#### 3.1 临界阈值检测

```python
【模块】src/phase_transition.py

【阈值判据】Percolation Phase Transition
农户i采纳绿色施肥需满足：

(Σ_j w_ij·a_j / max_k w_ik) + (P_i/P_max) - (C_i/C_max) ≥ Θ_c

三项平衡：
1. 社会网络项：邻居采纳情况
2. 政策激励项：补贴强度
3. 成本阻力项：经济负担

【临界点搜索】
扫描补贴强度 s ∈ [500, 3000]
对每个s，运行到稳态，记录采纳率ρ(s)

寻找梯度最大点：
s_c = argmax_s (dρ/ds)

【创新点7】临界阈值的网络修正
考虑超边后，阈值修正为：
Θ_c* = Θ_c · (1 - p_h)^(-κ)

κ = ln(⟨k_超⟩/⟨k⟩) / ln(1 + p_h)

p_h：超边比例
⟨k_超⟩：超边增强后的平均度
⟨k⟩：原始平均度

物理意义：超边降低了采纳阈值（合作社促进作用）
```

#### 3.2 Sigmoid曲线拟合

```python
【相变曲线模型】
ρ(s) = L / (1 + exp(-k(s - s_0)))

参数：
L：饱和采纳率
k：陡峭程度（相变强度）
s_0：临界补贴强度（拐点）

【拟合优度】
R² > 0.95 表明显著的S型相变

【创新点8】非线性政策响应
- s < s_c：政策无效区（低效投入）
- s ≈ s_c：政策突变区（高杠杆效应）
- s > s_c：政策饱和区（边际递减）

为政策制定提供科学依据
```

#### 3.3 网络传播效应分析

```python
【Hub节点识别】
k_max = max(degree(G))
hub_nodes = {i : degree(i) > 0.9 × k_max}

【扩散速度理论】
v ∝ √(γ·k_max)

γ：度分布指数
k_max：最大度数

【实验验证】
1. 让Hub节点率先采纳
2. 记录采纳率随时间演化
3. 计算实际扩散速度v_实际
4. 与理论预测对比

【创新点9】Hub节点的双重作用
- 正向：快速传播绿色施肥
- 负向：如果Hub节点不采纳，形成"意见领袖阻碍"
```
