# 🔄 完整项目运作流程

## 📋 总览

本项目是一个**复杂网络农户采纳行为模型**，通过模拟农户在社交网络中的决策过程，研究绿色施肥技术的扩散机制。

**核心流程：** 数据生成 → 网络构建 → 动力学仿真 → 相变分析 → 强化学习优化 → 结果可视化

---

## 🚀 完整运行流程

### 入口：`run_experiment.py`

```bash
python run_experiment.py --mode all
```

**运行模式：**
- `baseline`: 仅基线仿真
- `rl`: 仅强化学习训练
- `compare`: 仅对比分析
- `all`: 全部运行（默认）

## 📊 第一阶段：基线仿真（Baseline Simulation）

### 步骤1：生成农户数据 ⭐

**输入：**
- `config.yaml` 配置文件
- 参数：农户数量、属性分布参数等

**执行：**
```python
generator = FarmerDataGenerator(self.config)
farmer_df = generator.generate_farmer_attributes()
```

**处理过程：**
1. 生成基础属性：
   - 经济水平（正态分布：μ=50000, σ=20000）
   - 教育程度（0-3级）
   - 风险偏好（正态分布：μ=0.5, σ=0.2）
   - 环境意识（正态分布：μ=0.5, σ=0.25）
   - 短视性（均匀分布：U(0.1, 0.9)）

2. 生成三维质量：
   - 肥力（正态分布：N(0.5, 0.1²)）
   - 颗粒结构（正态分布：N(0.5, 0.1²)）
   - 生物活性（正态分布：N(0.5, 0.1²)）
   - 综合质量（加权平均）

3. 生成新维度：
   - 子女教育优先权（基于教育、经济水平）
   - 农民话语权（基于教育、经济水平）

4. 生成空间位置：
   - x, y, z坐标（单位球内均匀分布，用于网络构建）

5. 生成政策属性：
   - 政策感知度（均匀分布：U(0.3, 0.7)）
   - 培训访问权限（二项分布：30%概率）

6. 初始采纳状态：
   - 5%农户随机初始采纳绿色施肥

**输出：**
- `farmer_df`: pandas DataFrame（300行 × 19列）
- 保存到：`data/farmer_attributes.csv`

**数据结构：**
```
farmer_id | economic_level | education | risk_tolerance | ... | fertility | soil_structure | ... | x | y | z
```

### 步骤2：构建复杂网络 ⭐

**输入：**
- `farmer_df`（来自步骤1）
- `config.yaml` 网络参数

**执行：**
```python
builder = HyperbolicNetworkBuilder(self.config, farmer_df)
G = builder.build_scale_free_network()  # 构建无标度网络
builder.add_family_village_links(G)      # 添加家庭-村庄链接
hyperedges = builder.add_hyperedges(G) # 添加超边（合作社）
```

**处理过程：**
1. **构建无标度网络**（基于双曲几何）：
   - 使用农户的(x, y, z)坐标
   - 根据双曲距离连接节点
   - 生成幂律度分布（γ=2.5）

2. **添加家庭-村庄链接**：
   - 基于空间位置添加局部连接
   - 增加聚类系数

3. **添加超边（合作社）**：
   - 15%农户加入合作社
   - 合作社内部完全连接

**输出：**
- `G`: NetworkX图对象（300节点，~1200边）
- `hyperedges`: 超边字典
- 网络指标：
  - 平均度数：~8
  - 聚类系数：~0.03
  - 度分布：幂律（γ=2.5）
- 保存到：`data/network_edges.txt`

**网络结构：**
```
节点：300个农户
边：~1200条（普通边）
超边：~45个合作社（每个合作社3-10个农户）
```

### 步骤3：运行动力学仿真 ⭐

**输入：**
- `G`（网络，来自步骤2）
- `farmer_df`（数据，来自步骤1）
- `hyperedges`（超边，来自步骤2）
- `config.yaml` 动力学参数

**执行：**
```python
dynamics = AdoptionDynamics(self.config, G, farmer_df, hyperedges)

for t in range(60):  # 60个时间步
    dynamics.step_decision_update(subsidy, use_threshold=False)  # 更新决策
    dynamics.update_land_quality(dt=1.0)                        # 更新质量
    state = dynamics.get_system_state()                          # 记录状态
```

**处理过程（每个时间步）：**

#### 3.1 决策更新（`step_decision_update`）

对每个农户：
1. **计算TPB效用**（单位：元）：
   - 政策激励项：`S_policy = subsidy × perception × (1 + training × 0.2)`
   - 社会网络项：`S_social = neighbor_adoption_rate × 1000元`
   - 成本项：`C_cost = cost_green - subsidy`（绿色）或 `cost_traditional`（传统）
   - 质量收益项：`Q_benefit = Q_normalized × 5000元`
   - 总效用：`U = θ_p×S_policy + θ_s×S_social - θ_e×C_cost + θ_q×Q_benefit`

2. **Logit决策**：
   - 采纳概率：`P(adopt) = 1 / (1 + exp(-τ×U + ε))`
   - 随机采样决定是否采纳

3. **更新动态属性**：
   - 教育优先权：采纳时提升，不采纳时下降
   - 话语权：采纳时提升（受全局采纳率影响），不采纳时下降

#### 3.2 质量更新（`update_land_quality`）

对每个农户的三个质量维度：
1. **肥力演化**：
   - 绿色施肥：`dF/dt = α_F × (1-F) - γ_F × F + δ_F + D_F × 扩散项`
   - 传统施肥：`dF/dt = -β_F × F - γ_F × F + δ_F + D_F × 扩散项`

2. **颗粒结构演化**：
   - 类似肥力，但参数不同

3. **生物活性演化**：
   - 类似肥力，但参数不同

4. **邻居扩散**：
   - 质量从高质量邻居向低质量农户扩散

**输出（每个时间步）：**
- 系统状态字典：
  ```python
  {
      'adoption_rate': 0.35,           # 采纳率
      'global_Q': 0.52,                # 全局质量
      'mean_fertility': 0.48,          # 平均肥力
      'mean_soil_structure': 0.45,    # 平均颗粒结构
      'mean_biological_activity': 0.50, # 平均生物活性
      'mean_education_priority': 0.55,  # 平均教育优先权
      'mean_political_voice': 0.42,     # 平均话语权
      'mean_myopia': 0.50               # 平均短视性
  }
  ```

**最终输出：**
- `baseline_history.csv`：60行 × 4列（time, adoption_rate, global_Q, mean_land_quality）
- 最终采纳率：例如 0.35（35%）
- 最终全局质量：例如 0.52

### 步骤4：相变与临界阈值分析 ⭐

**输入：**
- 新的`farmer_df`（重新生成）
- `G`（网络，来自步骤2）
- `hyperedges`（超边，来自步骤2）
- 补贴范围：[500, 3000]元

**执行：**
```python
analyzer = PhaseTransitionAnalyzer(self.config)
subsidy_range = np.linspace(500, 3000, 20)  # 20个补贴值
phase_result = analyzer.analyze_percolation_threshold(dynamics_phase, subsidy_range)
```

**处理过程（对每个补贴值）：**

1. **重置系统**：
   - 所有农户初始为未采纳状态
   - 5%随机初始采纳

2. **运行到收敛**（50步）：
   - 更新决策
   - 更新质量

3. **计算序参数**：
   - 采纳率：`ρ = 采纳农户数 / 总农户数`
   - 全局质量：`Q = 平均质量值`
   - **巨大连通分量**：`P = 最大连通分量大小 / 总农户数` ⭐

4. **检测临界点**：
   - 采纳率临界点：梯度最大处
   - 逾渗临界点：巨大连通分量梯度最大处

**输出：**
- `phase_result` 字典：
  ```python
  {
      'subsidy_values': [500, 526, ..., 3000],        # 20个补贴值
      'adoption_rates': [0.12, 0.13, ..., 0.78],     # 对应采纳率
      'giant_component_sizes': [0.0, 0.0, ..., 0.65], # 巨大连通分量大小
      'global_qualities': [0.45, 0.46, ..., 0.68],    # 全局质量
      'critical_subsidy': 1650.5,                      # 采纳率临界补贴
      'critical_threshold': 0.39,                        # 采纳率临界阈值
      'percolation_subsidy': 1420.0,                   # 逾渗临界补贴
      'percolation_threshold': 0.25                     # 逾渗临界巨大连通分量
  }
  ```

5. **拟合Sigmoid曲线**：
   - 拟合采纳率相变曲线
   - 计算R²值

**输出：**
- `baseline_summary.json`：包含所有关键指标

## 🤖 第二阶段：强化学习训练（RL Training）

### 步骤1：初始化多智能体系统 ⭐

**输入：**
- `G`（网络，来自基线仿真的步骤2）
- `farmer_df`（数据，来自基线仿真的步骤1）
- `hyperedges`（超边，来自基线仿真的步骤2）
- `config.yaml` RL参数

**执行：**
```python
rl_system = MultiAgentSystem(self.config, G, farmer_df, hyperedges)
```

**处理过程：**
1. **为每个农户创建DQN智能体**：
   - 状态空间：12维
   - 动作空间：2维（传统/绿色）
   - 网络架构：12 → 64 → 64 → 2

2. **初始化网络参数**：
   - 学习率：0.001
   - 折扣因子：个性化（基于短视性）
   - 探索率：1.0（初始）

3. **创建经验回放池**：
   - 容量：5000条经验

**输出：**
- `MultiAgentSystem`对象：包含300个`FarmerAgent`

---

### 步骤2：训练循环 ⭐

**执行：**
```python
for episode in range(50):  # 50个训练轮
    for step in range(60):  # 每轮60步
        # 1. 获取状态
        # 2. 选择动作
        # 3. 执行动作
        # 4. 计算奖励
        # 5. 存储经验
        # 6. 更新网络
```

**处理过程（每个时间步）：**

#### 2.1 获取状态（`get_state_vector`）

对每个农户，构建12维状态向量：
```python
s = [
    economic_level / 100000,      # s₁: 归一化经济水平
    social_influence,             # s₂: 社会影响力权重
    policy_perception,            # s₃: 政策感知
    risk_tolerance,               # s₄: 风险偏好
    fertility,                    # s₅: 肥力
    soil_structure,               # s₆: 颗粒结构
    biological_activity,          # s₇: 生物活性
    neighbor_adoption,            # s₈: 邻居采纳率
    global_Q,                     # s₉: 全局质量
    myopia,                       # s₁₀: 短视性
    education_priority,           # s₁₁: 子女教育优先权
    political_voice              # s₁₂: 农民话语权
]
```

#### 2.2 选择动作（`select_action`）

ε-greedy策略：
- 探索（概率ε）：随机选择动作
- 利用（概率1-ε）：选择Q值最大的动作

#### 2.3 执行动作

更新农户的采纳状态：
- 动作0：传统施肥
- 动作1：绿色施肥

#### 2.4 计算奖励（`compute_reward`）

```python
R = λ₁×R_economic + λ₂×R_policy + λ₃×R_social + λ₄×R_education + λ₅×R_voice
```

其中：
- `R_economic`：经济效益（考虑三维质量）
- `R_policy`：政策补贴
- `R_social`：社会声誉
- `R_education`：子女教育优先权奖励
- `R_voice`：农民话语权奖励

#### 2.5 存储经验

存储到经验回放池：
```python
(state, action, reward, next_state, done)
```

#### 2.6 更新网络

从经验回放池采样128条经验：
1. 计算TD误差
2. 反向传播更新Q网络
3. 每5步更新目标网络

**输出（每个episode）：**
- 训练历史：
  ```python
  {
      'episode': 1,
      'adoption_rates': [0.05, 0.08, ..., 0.45],  # 60步的采纳率
      'global_Q': [0.50, 0.51, ..., 0.58],         # 60步的全局质量
      'mean_reward': 125.3,                        # 平均奖励
      'epsilon': 0.99                              # 当前探索率
  }
  ```

**最终输出：**
- `rl_training_history.csv`：50行 × 多列
- `rl_summary.json`：训练摘要
- 训练好的模型：`models/dqn_model_episode_50.pth`
