## 8. 常见问题

### Q1: 找不到Python命令
**问题：** `'python' is not recognized as an internal or external command`

**解决方案：**
1. 检查Python是否安装：
   ```bash
   python --version
   ```
2. 如果未安装，参考 `安装Python指南.txt`
3. 如果已安装但找不到，运行：
   ```bash
   自动查找Python.bat
   ```

---

### Q2: 依赖安装失败
**问题：** `pip install` 报错

**解决方案：**
```bash
# 升级pip
python -m pip install --upgrade pip

# 使用国内镜像
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或者逐个安装
pip install numpy
pip install pandas
pip install matplotlib
# ... 等等
```

---

### Q3: 运行速度太慢
**问题：** 完整实验需要1小时以上

**解决方案：**
使用快速配置：
```bash
python run_experiment.py --config config_fast.yaml
```

或者调整参数（编辑 `config.yaml`）：
```yaml
network:
  num_farmers: 100  # 减少农户数量

simulation:
  num_episodes: 20  # 减少训练轮数
  max_steps: 30     # 减少每轮步数
```

---

### Q4: 中文显示乱码
**问题：** 图表中的中文显示为方框

**解决方案：**
1. 安装中文字体（SimHei）
2. 或者修改 `src/visualization.py`：
   ```python
   plt.rcParams['font.sans-serif'] = ['Microsoft YaHei', 'SimHei']
   ```

---

### Q5: 内存不足
**问题：** `MemoryError` 或程序崩溃

**解决方案：**
减少参数规模（编辑 `config.yaml`）：
```yaml
network:
  num_farmers: 100  # 从300减少到100

rl:
  memory_size: 2000  # 从5000减少到2000
  batch_size: 64     # 从128减少到64
```

---

### Q6: 图表生成失败
**问题：** 某些图表没有生成

**解决方案：**
```bash
# 单独生成基础图表
python -c "from src.visualization import Visualizer; vis = Visualizer(); vis.plot_network(...)"

# 单独生成补充图表
python generate_supplementary_figures.py
```
