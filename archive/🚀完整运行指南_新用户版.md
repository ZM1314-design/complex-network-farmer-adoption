## 📝 方法二：手动分步运行

**⚠️ 前提条件**：Python必须已添加到系统PATH环境变量中，否则无法使用`python`命令。

**如何检查**：打开命令行，输入`python --version`，如果显示版本号则说明已配置好；如果显示"不是内部或外部命令"，则需要先配置PATH。

**如果Python未添加到PATH**：
- **方法A**：重新安装Python，安装时勾选"Add Python to PATH"
- **方法B**：手动添加Python到PATH（见下方说明）
- **方法C**：使用完整路径运行Python（如：`C:\Python39\python.exe`）

如果自动运行失败，可以手动执行以下步骤：

### 步骤1：打开命令行

- **Windows**：按 `Win + R`，输入 `cmd`，回车
- 或者：在项目文件夹中，按住 `Shift` 右键，选择"在此处打开PowerShell窗口"

### 步骤2：进入项目目录

```bash
cd 复杂网络
```

（如果文件夹名不同，请改为实际文件夹名）

### 步骤3：检查Python环境

```bash
python --version
```

**应该显示**：`Python 3.x.x`（3.8或更高）

如果显示"不是内部或外部命令"，说明Python未安装或未添加到PATH。

### 步骤4：安装依赖包

```bash
pip install -r requirements.txt
```

**如果pip命令失败，尝试：**
```bash
python -m pip install -r requirements.txt
```

**如果下载慢，可以使用国内镜像：**
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**等待安装完成**（可能需要5-10分钟）

### 步骤5：验证安装

```bash
python test_environment.py
```

**应该显示**：所有依赖检查通过 ✅

### 步骤6：运行完整实验

```bash
python run_complete_experiment.py
```

**或者使用批处理文件：**
```bash
一键运行完整实验.bat
```

**等待运行完成**（可能需要30-60分钟）

## ⚠️ 常见问题解决

### 问题1：`pip` 命令不存在

**解决方案：**
```bash
python -m pip install -r requirements.txt
```

### 问题2：下载依赖包很慢

**解决方案：使用国内镜像**
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题3：安装某个包失败（如torch）

**解决方案：**
- **Windows + CPU**：
  ```bash
  pip install torch --index-url https://download.pytorch.org/whl/cpu
  ```
- **Windows + GPU（如果有NVIDIA显卡）**：
  ```bash
  pip install torch --index-url https://download.pytorch.org/whl/cu118
  ```

### 问题4：内存不足

**解决方案：**
- 关闭其他程序
- 修改 `config.yaml` 中的参数：
  - `num_farmers: 300` → `num_farmers: 100`（减少农户数）
  - `num_runs: 5` → `num_runs: 1`（减少运行次数）

### 问题5：运行时间太长

**解决方案：**
- 修改 `config.yaml` 中的参数：
  - `num_episodes: 200` → `num_episodes: 50`（减少训练轮数）
  - `max_steps: 60` → `max_steps: 30`（减少仿真步数）
  - `num_farmers: 300` → `num_farmers: 100`（减少农户数）

### 问题6：Python版本太低

**解决方案：**
- 需要Python 3.8或更高版本
- 下载安装：https://www.python.org/downloads/
- 安装时勾选"Add Python to PATH"

### 问题7：`python` 命令找不到（未添加到PATH）

**解决方案A：重新安装Python（推荐）**
1. 卸载当前Python
2. 重新下载安装：https://www.python.org/downloads/
3. **安装时务必勾选"Add Python to PATH"**

**解决方案B：手动添加Python到PATH**
1. 找到Python安装路径（通常在 `C:\Python39\` 或 `C:\Users\用户名\AppData\Local\Programs\Python\Python39\`）
2. 右键"此电脑" → "属性" → "高级系统设置" → "环境变量"
3. 在"系统变量"中找到"Path"，点击"编辑"
4. 点击"新建"，添加Python安装路径（如：`C:\Python39\`）
5. 再添加Scripts路径（如：`C:\Python39\Scripts\`）
6. 点击"确定"保存
7. **重新打开命令行**，再次尝试`python --version`

**解决方案C：使用完整路径（临时方案）**
- 如果Python安装在 `C:\Python39\`，使用：
  ```bash
  C:\Python39\python.exe --version
  C:\Python39\python.exe -m pip install -r requirements.txt
  C:\Python39\python.exe run_complete_experiment.py
  ```
