# 📋 完整手动运行步骤 - 详细版

**适用场景**：只有Python环境，需要手动执行每一步

**预计时间**：40-70分钟（安装依赖5-10分钟 + 运行实验30-60分钟）

---

## ⚠️ 前提条件检查

### 步骤0.1：检查Python是否已安装

1. **打开命令行**：
   - 按 `Win + R` 键
   - 输入 `cmd`
   - 按回车

2. **检查Python版本**：
   ```bash
   python --version
   ```

3. **判断结果**：
   - ✅ **如果显示**：`Python 3.x.x`（如 `Python 3.9.7`）
     - 说明Python已安装且已添加到PATH
     - **继续下一步**
   
   - ❌ **如果显示**：`'python' 不是内部或外部命令，也不是可运行的程序或批处理文件。`
     - 说明Python未安装或未添加到PATH
     - **必须先解决此问题**（见下方"Python PATH配置"）

---

### 步骤0.2：Python PATH配置（如果需要）

#### 情况A：Python未安装

1. **下载Python**：
   - 访问：https://www.python.org/downloads/
   - 下载最新版本的Python 3.8或更高版本

2. **安装Python**：
   - 运行下载的安装程序
   - **⚠️ 重要**：勾选 "Add Python to PATH"（添加到PATH）
   - 点击"Install Now"
   - 等待安装完成

3. **验证安装**：
   - **重新打开命令行**（重要！）
   - 输入：`python --version`
   - 应该显示版本号

#### 情况B：Python已安装但未添加到PATH

**方法1：重新安装（推荐）**
- 按照情况A的步骤重新安装，勾选"Add Python to PATH"

**方法2：手动添加PATH**
1. **找到Python安装路径**：
   - 常见位置：
     - `C:\Python39\`
     - `C:\Users\你的用户名\AppData\Local\Programs\Python\Python39\`
   - 如果找不到，在文件资源管理器中搜索 `python.exe`

2. **打开环境变量设置**：
   - 右键点击"此电脑"（或"我的电脑"）
   - 选择"属性"
   - 点击"高级系统设置"
   - 点击"环境变量"按钮

3. **编辑PATH变量**：
   - 在"系统变量"区域找到"Path"
   - 选中"Path"，点击"编辑"
   - 点击"新建"
   - 输入Python安装路径（如：`C:\Python39\`）
   - 再点击"新建"，输入Scripts路径（如：`C:\Python39\Scripts\`）
   - 点击"确定"保存所有窗口

4. **验证配置**：
   - **关闭所有命令行窗口**
   - **重新打开命令行**
   - 输入：`python --version`
   - 应该显示版本号

## 📝 完整运行步骤

### 步骤1：打开命令行并进入项目目录

1. **打开命令行**：
   - 按 `Win + R`
   - 输入 `cmd`
   - 按回车

2. **进入项目目录**：
   ```bash
   cd 复杂网络
   ```
   
   **注意**：
   - 如果项目文件夹不在当前目录，需要先切换到项目所在位置
   - 例如：`cd C:\Users\你的用户名\Desktop\复杂网络`
   - 或者：在项目文件夹中，按住 `Shift` 键，右键点击空白处，选择"在此处打开PowerShell窗口"或"在此处打开命令窗口"

3. **验证当前目录**：
   ```bash
   dir
   ```
   
   **应该看到**：`config.yaml`、`requirements.txt`、`run_complete_experiment.py` 等文件

---

### 步骤2：检查Python环境

1. **检查Python版本**：
   ```bash
   python --version
   ```
   
   **应该显示**：`Python 3.x.x`（3.8或更高）

2. **检查pip是否可用**：
   ```bash
   python -m pip --version
   ```
   
   **应该显示**：`pip x.x.x from ...`

3. **如果以上命令都失败**：
   - 返回"步骤0.2"配置Python PATH
   - 或使用完整路径（如：`C:\Python39\python.exe -m pip --version`）

---

### 步骤3：安装依赖包

1. **安装所有依赖**：
   ```bash
   pip install -r requirements.txt
   ```
   
   **如果pip命令失败，使用：**
   ```bash
   python -m pip install -r requirements.txt
   ```

2. **如果下载很慢，使用国内镜像**：
   ```bash
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```
   
   或：
   ```bash
   python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   ```

3. **等待安装完成**：
   - 这个过程可能需要 **5-10分钟**
   - 会看到很多包的安装信息
   - 最后应该显示类似：`Successfully installed ...`

4. **如果某个包安装失败**（特别是torch）：
   
   **Windows + CPU版本**：
   ```bash
   pip install torch --index-url https://download.pytorch.org/whl/cpu
   ```
   
   **Windows + GPU版本**（如果有NVIDIA显卡）：
   ```bash
   pip install torch --index-url https://download.pytorch.org/whl/cu118
   ```
   
   然后继续安装其他依赖：
   ```bash
   pip install -r requirements.txt
   ```

---

### 步骤4：验证环境配置

1. **运行环境检查脚本**：
   ```bash
   python test_environment.py
   ```
   
   **应该显示**：
   ```
   ✓ Python版本: 3.x.x
   ✓ Python路径: ...
   检查依赖包:
   ✓ numpy - 已安装
   ✓ scipy - 已安装
   ✓ networkx - 已安装
   ✓ torch - 已安装
   ...（所有包都显示"已安装"）
   ✓ 所有依赖都已安装！环境配置完成！
   ```

2. **如果有包未安装**：
   - 会显示哪些包缺失
   - 按照提示安装缺失的包
   - 例如：`pip install 包名`

---

### 步骤5：确认配置文件

1. **检查配置文件是否存在**：
   ```bash
   dir config.yaml
   ```
   
   **应该显示**：`config.yaml` 文件

2. **（可选）检查训练轮数设置**：
   - 打开 `config.yaml` 文件
   - 找到 `num_episodes: 200`（应该已经是200）
   - 如果不是200，修改为200

---

### 步骤6：运行完整实验

1. **运行实验脚本**：
   ```bash
   python run_complete_experiment.py
   ```
   
   **如果python命令失败，使用完整路径**：
   ```bash
   C:\Python39\python.exe run_complete_experiment.py
   ```
   （将路径改为你的Python实际安装路径）

2. **运行过程说明**：
   - 会看到很多输出信息
   - 包括：
     - 生成农户数据
     - 构建网络
     - 运行基线仿真
     - 运行强化学习训练（200轮）
     - 生成各种图表
   - **这个过程需要 30-60分钟**，请耐心等待
   - 不要关闭命令行窗口

3. **运行成功的标志**：
   - 最后会显示类似：
     ```
     [OK] 所有图表已生成
     [OK] 实验完成
     ```
   - 或者没有错误信息，正常结束

4. **如果运行失败**：
   - 查看错误信息
   - 常见问题见下方"故障排除"

---

### 步骤7：验证运行结果

1. **检查图表文件夹**：

## 🔧 故障排除

### 问题1：`python` 命令找不到

**解决方案**：
- 返回"步骤0.2"配置Python PATH
- 或使用完整路径运行Python

### 问题2：`pip` 命令找不到

**解决方案**：
```bash
python -m pip install -r requirements.txt
```

### 问题3：依赖包下载很慢

**解决方案**：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题4：torch安装失败

**解决方案**：
- CPU版本：
  ```bash
  pip install torch --index-url https://download.pytorch.org/whl/cpu
  ```
- GPU版本：
  ```bash
  pip install torch --index-url https://download.pytorch.org/whl/cu118
  ```

### 问题5：内存不足

**解决方案**：
1. 关闭其他程序释放内存
2. 修改 `config.yaml`：
   - `num_farmers: 300` → `num_farmers: 100`
   - `num_runs: 5` → `num_runs: 1`

### 问题6：运行时间太长

**解决方案**：
- 这是正常的，200轮训练需要30-60分钟
- 如果想加快，可以修改 `config.yaml`：
  - `num_episodes: 200` → `num_episodes: 50`
  - `max_steps: 60` → `max_steps: 30`

### 问题7：找不到某个文件

**解决方案**：
1. 确认当前目录正确：`cd 复杂网络`
2. 列出文件：`dir` 查看所有文件
3. 确认文件确实存在
