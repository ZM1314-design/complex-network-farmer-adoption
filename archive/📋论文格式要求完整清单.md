## 📄 三、中文摘要页

### 页眉：
- **内容**：摘要（宋体小五）
- **位置**：页面顶部

### 标题：
- **内容**："摘要"
- **格式**：小二黑体加粗居中，无空行

### 正文：
- **字体**：五号宋体/Time New Roman 不加粗
- **格式**：段首空两格，单倍行距
- **字数**：约300字
- **内容**：目的、研究方法、结果、结论

### 关键词：
- **位置**：摘要与关键词之间要空一行
- **格式**："关键词"（黑体五号不加粗顶格写）
- **内容**：XX, XX, XX（五号宋体/Time New Roman 不加粗）
- **数量**：3-5个

### 页码：
- **格式**：罗马数字（I, II, III...）
- **样式**：不带两侧横线

### 页码：

## 📑 五、目录页

### 页眉：
- **内容**：目录（宋体小五）
- **位置**：页面顶部

### 标题：
- **内容**："目录"
- **格式**：小二黑体不加粗居中，段前0.5行段后0.5行

### 内容格式：

- **三级标题**（如"1.2.1 国外研究现状"）：
  - 五号宋体不加粗
  - 缩进1.7字符

### 排版要求：
- **行距**：单倍行距
- **对齐**：两端与页边距对齐，页码右对齐
- **页码格式**：新罗马五号，不带横杠（区别于正文页码）

### 页码：
- **格式**：罗马数字（I, II, III...）
- **样式**：不带两侧横线

## 📖 六、正文

### 页面设置：
- **页边距**：上3.8cm，下4.8cm，左2.4cm，右2.4cm
- **页眉**：2.8cm
- **页脚**：3.8cm
- **纸型**：A4
- **文档网格**：每行38个字，每页38行
- **行距**：单倍行距

### 页眉：
- **内容**：论文题目（宋体小五）
- **位置**：居中
- **样式**：下加粗、细双线（粗线在上，宽3磅）

### 页脚：
- **内容**：页码
- **格式**："-n-"（新罗马小五）
- **位置**：页面底端居中
- **样式**：阿拉伯数字，两侧带横线

### 标题格式：

#### 一级标题（如"1 绪论"）：
- **字体**：小二黑体不加粗
- **位置**：顶格
- **间距**：段前0.5行，段后0.5行
- **编号**：阿拉伯数字（1, 2, 3...）

#### 二级标题（如"1.1 研究目的意义"）：
- **字体**：小三黑体不加粗
- **位置**：首行缩进1字符
- **间距**：无段前段后间距
- **编号**：1.1, 1.2, 1.3...

#### 三级标题（如"1.2.1 国外研究现状"）：
- **字体**：四号黑体不加粗
- **位置**：首行缩进1.5字符
- **间距**：无段前段后间距
- **编号**：1.2.1, 1.2.2...

#### 四级标题（如"1.2.1.1"）：
- **字体**：小四号黑体不加粗
- **编号**：1.2.1.1...

**注意**：一般不超过4级标题

### 正文格式：
- **字体**：五号宋体/Time New Roman 不加粗
- **格式**：段首空两格，单倍行距
- **标点**：
  - 数字和单位之间空一格
  - 。、℃和%与数字之间不加空格

### 章节要求：
- **新章节**：另起一页
- **章节小结**：除第一张绪论及最后一章结论外，其余章节最后应有本章小结

## 📊 七、图表格式

### 图（Figure）：

#### 基本要求：
- **位置**：居中显示
- **背景**：截图背景为白色
- **图名格式**：五号宋体/Time New Roman 不加粗居中
- **图号格式**：图X-X（按章节顺序），图号与图名之间有空格

#### 图中字体：
- **字体**：宋体/Time New Roman 不加粗
- **字号**：经过缩放后的图中的文字及数字字号不大于正文
- **要求**：必须清楚标出横、纵坐标题目及单位

#### 多个图片：
- 如果一个图名下有多个图片，需标注(a)、(b)并表明内容
- 在文中解释说明内容时需注明图X-X(a)
- 多个图片时，应保证图片大小尽可能一致

#### 示例：
```
图2-1 不同超声频率下产氢曲线
```

### 表（Table）：

#### 基本要求：
- **类型**：三线表（toprule, midrule, bottomrule）
- **表头格式**：五号宋体/Time New Roman 不加粗居中
- **表号格式**：表X-X，表号与表名之间有空格
- **表名格式**：五号宋体/Time New Roman 不加粗居中

#### 表格样式：
- **顶线和底线**：1.5磅
- **中间线**：0.75磅
- **表内文字**：五号宋体不加粗居中
- **内容过多**：可缩小到小五号

#### 多页表格：
- 续页右上角标注"续上表"

#### 间距：
- 表格与正文之间空一行

#### 示例：
```
表2-1 铁路客运售票主要业务
```

## 🔢 八、公式格式

### 基本要求：
- **位置**：居中
- **字体**：宋体/Time New Roman 不加粗
- **字号**：五号（与正文字号相当）
- **变量**：使用斜体
- **常量和单位**：使用正体

### 编号格式：
- **格式**：(X-X)代表第X章第X个公式
- **位置**：右对齐
- **示例**：(2-1)表示第2章第1个公式

### 公式说明：
- **行距**：与正文相当
- **格式**：式中：变量名 ————— 说明，单位；

### 示例：
```
T_tq = Σ (i=0 to k) A_i n_e^i
                    (i = 0, 1, ..., k)        (2-1)

式中：T_tq ————— 发动机转矩，N·m；
      n_ei —— 发动机转速，r/min；
      k——多项式阶数；
      A_i————多项式系数。
```

## 📚 九、参考文献

### 标题：
- **内容**："参考文献"
- **格式**：小二黑体不加粗居中，段前段后0.5行

### 排序要求：
- 按照在文中引用的顺序排列

### 格式要求：
- **文献类型标识**：必须标注[M]、[J]、[D]等
  - [M] = 图书
  - [J] = 期刊
  - [D] = 学位论文
  - [C] = 会议录
  - [R] = 报告
  - [S] = 标准
  - [P] = 专利
  - 等等
- **页码**：必须标注参考的页码，如：123-456
- **标点符号**：
  - 中文参考文献：标点符号采用全角
  - 英文参考文献：标点符号为半角后加一空格
- **字体**：五号宋体/Time New Roman
- **标号格式**：标号与参考文献中间不加空格，换行后顶格书写

### 数量要求：
- **外文文献**：不少于5篇
- **近五年文献**：不少于5篇

### 示例：
```
[1] 王金武,人因工程学[M].哈尔滨:东北农业大学出版社,2007: 190-199.
[2] 孟雅哲,循环流化床锅炉燃烧系统建模及预测控制仿真研究[D].昆明:昆明理工大学,2012:39-42.
[3] Liakatas A, Clark J A, Monteith J L. Measurements of the heat balance under plastic mulches[J]. Agricultural and Forest Meteorology, 1986, 36(1): 23-31.
```

### 标准：
- 参考格式：GB/T 7714-2015

## 📌 关键注意事项

1. **页码区分**：
   - 摘要、目录：罗马数字，不带横线（I, II, III...）
   - 正文：阿拉伯数字，带横线（-1-, -2-, -3...）

2. **页眉区分**：
   - 摘要页：页眉为"摘要"或"Abstract"
   - 目录页：页眉为"目录"
   - 正文页：页眉为"论文题目"

3. **字体区分**：
   - 中文：宋体
   - 英文/数字：Time New Roman
   - 标题：黑体
   - 正文：宋体/Time New Roman

4. **间距要求**：
   - 摘要与关键词之间：空一行
   - 表格与正文之间：空一行
   - 一级标题：段前0.5行，段后0.5行

5. **打印要求**：
   - 从英文摘要后开始正反面打印
   - 封面采用白色+网格式
