## 常见问题

### Q1: 提示 "No module named 'xxx'"

**解决方案：**
```cmd
pip install -r requirements.txt
```

---

### Q2: 提示 "FileNotFoundError: config.yaml"

**原因：** 当前目录不对

**解决方案：**
```cmd
cd C:\Users\zhaom\Desktop\复杂网络
python run_experiment.py --mode baseline --config config.yaml
```

---

### Q3: 运行很慢

**原因：** 使用的是平衡配置（300农户，60步，50轮）

**解决方案（快速测试）：**
```cmd
python run_experiment.py --mode baseline --config config_fast.yaml
```

---

### Q4: 图表中文乱码

**原因：** 缺少中文字体

**解决方案：**
1. 检查是否安装了 SimHei 字体
2. 或修改 `src/visualization.py` 第13行：
```python
plt.rcParams['font.sans-serif'] = ['Microsoft YaHei', 'SimHei', 'DejaVu Sans']
```

---

### Q5: Unicode编码错误

**原因：** Windows终端默认使用gbk编码

**解决方案：**
已修复所有`✓`字符为`[OK]`，不应再出现此问题。

如果仍有问题：
```cmd
chcp 65001
python run_experiment.py --mode baseline --config config.yaml
```

---

### Q6: 想重新生成某张图表

**重新生成基础图表：**
```cmd
python run_experiment.py --mode baseline --config config.yaml
```

**重新生成补充图表：**
```cmd
python generate_supplementary_figures.py
```

---

### Q7: 想修改参数

**编辑配置文件：**
```cmd
notepad config.yaml
```

**主要参数：**
- `num_farmers`: 农户数量（默认300）
- `max_steps`: 仿真步数（默认60）
- `num_episodes`: RL训练轮数（默认50）
- `subsidy_base`: 基础补贴（默认1200元）

修改后保存，重新运行：
```cmd
python run_experiment.py --mode baseline --config config.yaml
```
