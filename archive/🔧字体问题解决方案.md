# 🔧 字体问题解决方案

## 问题原因

错误：`Font C70/rm/m/n/12/01f3=unisong01f3 at 12.0pt not loadable`

这是因为 `ctex` 包需要特定的中文字体文件，但MiKTeX可能没有自动安装。

---

## 解决方案（3种方法）

### 方法1：使用XeLaTeX编译（推荐，最简单）

XeLaTeX对中文支持更好，不需要额外的字体包。

**步骤：**

1. 打开 `test_advanced_xe.tex`（我已经创建了这个文件）
2. 按 `Ctrl+Shift+P`
3. 输入 "LaTeX: Build with recipe"
4. 选择 "xelatex"
5. 编译

**或者：**
- 右键点击文件
- 选择 "Build LaTeX project"
- 选择使用 xelatex

---

### 方法2：安装中文字体包

1. **打开MiKTeX Console**：
   - 开始菜单搜索 "MiKTeX Console"
   - 或运行命令：`miktex-console`

2. **更新包数据库**：
   - 点击 "Packages" 标签
   - 点击 "Update" 更新包列表

3. **安装字体包**：
   - 搜索 "ctex" 或 "xeCJK"
   - 安装相关的中文字体包

4. **重新编译** `test_advanced.tex`

---

### 方法3：使用系统字体（已修改）

我已经修改了 `test_advanced.tex`，使用系统字体。

**尝试：**
1. 重新编译 `test_advanced.tex`（`Ctrl+Alt+B`）
2. 如果还不行，使用方法1（XeLaTeX）

---

## 推荐方案

**最简单的方法：使用XeLaTeX**

1. 打开 `test_advanced_xe.tex`
2. 按 `Ctrl+Shift+P` → "LaTeX: Build with recipe" → 选择 "xelatex"
3. 编译

XeLaTeX对中文支持更好，通常不会有字体问题。
